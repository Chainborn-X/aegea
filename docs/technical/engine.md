# Engine

## Status

No engine has been selected yet.

## Evaluation Criteria

The engine should support:

- Top-down 2D action RPG movement and combat.
- Painterly environments and layered scenes.
- Tilemaps, hand-placed props, or a hybrid workflow.
- Fast iteration for exploration design.
- Cross-platform builds if the project scope requires them.
- Source control friendliness.
- Custom tools for content validation and production workflows.

## Prototype Questions

- How quickly can a coastline, cave, temple, and forest scene be built?
- Can the camera preserve player-is-small composition while keeping gameplay readable?
- Can the engine handle layered vegetation, water, lighting, and atmospheric effects?
- How comfortable is the workflow for non-programming content creation?
- How easy is it to build importers, validators, and custom editor tools?
