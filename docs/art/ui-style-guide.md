# UI Style Guide

## UI Goals

The interface should support exploration without crowding the world. It should feel crafted, ancient, and quiet, but remain fast and readable.

## Visual Direction

Use:

- Warm stone, faded pigment, bronze, linen, and sea-glass accents.
- Simple icon silhouettes inspired by artifacts, constellations, tools, and natural forms.
- Clear spacing and strong readability.
- Minimal decoration during exploration.

## HUD

The default HUD should be restrained. The world should dominate the screen.

Show only what the player needs moment to moment. Let menus carry deeper inventory, map, lore, and progression information.

## Map

The map should encourage memory and curiosity rather than completely solving exploration.

Possible approaches:

- Hand-drawn regional maps.
- Incomplete maps filled through travel.
- Landmarks marked by sketches rather than abstract icons.
- Notes from shrines, NPCs, or discoveries.

## Avoid

- Overly modern sci-fi panels.
- Loud mobile-game UI.
- Excessive quest markers.
- UI that explains every mystery immediately.
