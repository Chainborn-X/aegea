# Items

## Item Identity

Items should feel discovered, crafted, offered, found in ruins, or inherited from vanished cultures. Avoid generic loot clutter.

## Item Categories

- Relics: unique objects that change traversal, combat, or world interaction.
- Offerings: objects used at shrines, groves, graves, or sacred springs.
- Materials: regionally grounded resources for crafting and restoration.
- Lore objects: tablets, pottery shards, coins, charms, maps, and inscriptions.
- Consumables: simple, readable support items with strong visual identity.

## Naming

Names should be evocative but clear. Prefer names rooted in material, place, ritual, or mythic function.

Examples:

- Salt-Bright Lamp
- Laurel Offering
- Broken Harbor Key
- Bronze Reed Pin
- Asphodel Thread

## Avoid

- Excessive randomized loot.
- Generic fantasy item names.
- Inventory bloat that distracts from exploration.
