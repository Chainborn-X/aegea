# Progression

## Progression Philosophy

Progression should expand the player's relationship with the world. New abilities should make old places feel newly readable.

## Progression Types

- Traversal abilities.
- Relics and sacred tools.
- Combat techniques.
- Map knowledge.
- Regional blessings.
- Access to boats, lifts, bridges, doors, shrines, and hidden paths.

## Soft Gates

Prefer soft gates when possible:

- A dangerous route that becomes easier later.
- A visible path requiring a learned symbol.
- A cave reachable only when the tide is low.
- A mountain pass that requires warmth, light, or wind knowledge.

## Hard Gates

Use hard gates sparingly and make them feel diegetic:

- Temple seals.
- Broken mechanisms.
- Divine thresholds.
- Flooded passages.
- Collapsed bridges.

## Avoid

- Progression as stat inflation only.
- Checklist unlocks that do not change exploration.
- Overexplaining every gate in UI text.
