# Exploration Systems

## Discovery

The map begins incomplete. The player fills it through:

- climbing watchtowers
- buying sailor charts
- restoring ancient map stones
- listening to NPC rumors
- reading inscriptions
- following visible landmarks
- decoding songs, murals, and grave names

The game should avoid excessive quest logs. Guidance should come from:

- map annotations
- sketches
- short memory phrases
- NPC hints
- environmental clues
- visible distant landmarks

## Hidden Passages

Hidden routes should feel physically plausible and tied to relic knowledge.

Examples:

- cracked walls opened by the Gauntlet of Hephaestus
- moon doors visible only at night
- spirit bridges revealed with the Obol of Hades
- hearth flames that reveal old architecture
- tide caves accessible at low water
- wind gaps crossed with the Bronze-Wing Harness
- sun-shadow doors revealed by the Lyre of Dawn

## Ancient Maps

Ancient maps are partial and symbolic, not complete UI solutions.

Examples:

- a sea chart showing bells rather than islands
- a desert map based on shadows
- an Underworld map that changes when names are restored
- a sky map that overlays hidden Olympus fragments
- a hunter map drawn from animal migration instead of roads

## Forgotten Languages

Languages are progression systems. Learning a language should open routes, lore, puzzles, and alternate resolutions.

### Hearth-script

Domestic marks, recipes, family names, and household rites.

Use cases:

- restore homes
- understand village memories
- solve Hestian puzzles

### Temple Greek

Official divine history.

Use cases:

- read temple inscriptions
- identify contradictions
- decode public myths

### First Naming

Pre-divine symbolic language.

Use cases:

- access Primordial sanctums
- unlock secret ending context
- understand the original Pact

### River Cant

Names of the dead and ritual language of crossing.

Use cases:

- restore graves
- speak with shades
- cross Underworld routes safely

### Machine Glyphs

Daedalan engineering notation.

Use cases:

- activate ancient devices
- understand Meridian schematics
- solve machinery puzzles

## Treasure Hunting

Treasure should usually be unique.

Preferred rewards:

- relics
- songs
- heart vessels
- spirit vessels
- map fragments
- lost names
- weapon techniques
- ship upgrades
- myth fragments

Coins can exist, but meaningful treasure should change play or story.

## Relic Hunting

Relics are progression keys and identity markers.

Each relic must:

- open routes
- solve puzzles
- alter combat
- reveal memories
- connect to a god or Primordial

## Myth Hunting

Players collect conflicting versions of myths. Completing a myth set reveals the hidden truth behind a region.

Example set:

- "The Hero Slew the Lion."
- "The Lion Guarded the Gate."
- "The King Poisoned the Lion."
- "The Hero Wept Afterward."

Complete sets unlock:

- secret rooms
- alternate boss dialogue
- ending modifiers
- companion commentary
- regional state changes

## Secret Bosses

Secret bosses should be buried truths, not just optional combat checks.

Examples:

- **Gaia's Root-Heart**
- **Nyx's Unlit Child**
- **The Hero Who Lied**
- **The Automaton That Prays**
- **The Nameless First Pactbearer**

## Secret Endings

Secret ending requirements should reward curiosity and synthesis:

- restore all dead names
- complete myth sets
- find Primordial sanctums
- resolve companion arcs
- refuse simplified versions of history
