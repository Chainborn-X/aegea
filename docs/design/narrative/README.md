# Echoes of Olympus Narrative Design

This folder contains the production-facing narrative and world design foundation for **Echoes of Olympus**, a 2D top-down action-adventure RPG set in Aegea.

The goal of these documents is to convert the creative premise into usable production material: world history, story architecture, region and dungeon identity, exploration systems, item lore, enemy ecology, progression, and chapter structure.

Use these files with the existing repo pillars:

- `docs/vision` defines the emotional target.
- `docs/world` defines the broader Aegea setting.
- `docs/design` defines core game design.
- `docs/design/narrative` defines the specific story and content foundation for Echoes of Olympus.
- `docs/design/implementation` translates the foundation into buildable formats and milestones.

## Narrative Design Rules

- The world should feel ancient, alive, interconnected, and worth exploring.
- Every major place should contain a mystery.
- Every mystery should reveal a larger mystery.
- Lore should be discovered through place, object, ritual, memory, song, and contradiction before exposition.
- Progression should come from relics, knowledge, exploration, divine blessings, and mastery.
- Avoid quest-log-driven design, repetitive tasks, and exposition dumps.
- The player should feel like they are uncovering a forgotten world one meaningful secret at a time.
