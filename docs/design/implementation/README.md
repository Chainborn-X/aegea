# Implementation Planning

This folder translates the Echoes of Olympus narrative and world foundation into buildable production structure.

Use it to move from story intent to scenes, scenarios, systems, and milestones.

## Document Map

- [roadmap.md](roadmap.md): milestone plan from foundation audit through ending framework.
- [vertical-slice.md](vertical-slice.md): first playable slice, scoped around Elia and Hearthwood Shrine.
- [content-pipeline.md](content-pipeline.md): how regions, scenarios, scenes, flags, and content units should be authored.
- [quest-format.md](quest-format.md): reusable scenario format and example scenario.
- [scene-format.md](scene-format.md): reusable scene format and example Echo memory.

## Production Principle

Do not treat narrative as a layer added after mechanics. For Echoes of Olympus, discovery, memory, myth fragments, inscriptions, relic gates, and changed world states are gameplay systems.
