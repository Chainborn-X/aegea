# Architecture Generation Prompts

## Architecture Template

Design architecture for Aegea, a top-down action RPG set in a mythological Mediterranean world inspired by ancient Greece. Structure type: [TEMPLE / SHRINE / HARBOR / VILLAGE / BRIDGE / GATE / TOMB / CISTERN]. Region: [REGION]. The design should use ancient Mediterranean forms, cracked marble or limestone, faded pigment, bronze details, weathering, vines, moss, roots, flowers, and evidence of nature reclaiming civilization. Include top-down readability, traversal function, discovery hooks, and mythological symbolism. Not historical reconstruction, not generic medieval fantasy, not pristine, not grimdark.

## Required Output Fields

- Structure purpose.
- Silhouette.
- Materials.
- Age and weathering.
- Relationship to landscape.
- Traversal use.
- Secret or puzzle possibilities.
- Decorative motifs.
- Palette.
